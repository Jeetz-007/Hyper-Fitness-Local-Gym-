## Project Overview

This project is a fully responsive static website built for a real client (gym owner & trainer) to showcase:

- Gym services & pricing
- Trainer profile and experience
- Gallery & achievements
- Client reviews
- Contact & lead generation

The site is optimized for **performance, mobile devices, and SEO**, and deployed using **Netlify** with a custom domain.

---

## Features

- Responsive design (mobile, tablet, desktop)
- Sticky navigation bar with hamburger menu
- Hero section with call-to-action buttons
- About section with trainer profile & certifications
- Image gallery with slider
- Services & pricing section
- Client reviews slider
- Contact form powered by Netlify Forms
- Sticky WhatsApp contact button
- SEO & Open Graph meta tags
- Custom domain + HTTPS
